# 🔐 Personal Vault CLI

A command-line Personal Vault application built in Python that allows users to securely organize their personal notes.

This project was created as the **Phase 01 Capstone Project** of my Python learning journey. It combines multiple Python concepts into a single practical application.

---

## ✨ Features

- User Signup
- User Login
- Add Notes
- View Notes
- Edit Notes
- Delete Notes
- Search Notes using:
  - Custom keywords or Regular Expressions
  - Email detection
  - Phone number detection
  - IP address detection
  - Date detection
  - Monetary value detection
- Persistent data storage using JSON
- Input validation with Exception Handling

---

## 🧠 Concepts Practiced

- Functions
- Modules
- Lists & Dictionaries
- JSON
- File Handling
- Exception Handling
- Regular Expressions (Regex)
- String Formatting
- User Input
- Modular Programming
