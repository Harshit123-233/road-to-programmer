# 🎮 Hangman Game (Python)

A console-based Hangman game built with Python using Object-Oriented Programming principles.

This project was developed as part of my Python learning journey and focuses on writing clean, modular, and maintainable code rather than simply making the game work.

---

## ✨ Features

- 👤 User Authentication
  - Sign Up
  - Log In
  - Guest Mode

- 🎯 Four Difficulty Levels
  - Easy
  - Medium
  - Hard
  - Bonus (Programming & Tech words)

- 🏆 Persistent Leaderboard

- 💾 Player Progress Saving

- ❤️ Classic Hangman Gameplay
  - Guess one letter at a time
  - Guess the complete word
  - Six lives
  - ASCII Hangman drawing

- 🎖️ Score System
  - Easy → +5
  - Medium → +10
  - Hard → +15
  - Bonus → +25

- 👥 Guest Restrictions
  - One game only
  - Easy mode only
  - Progress is not saved

## 🚀 How to Run

Clone the repository:

```bash

```

Move into the project directory:

```bash
cd Hangman
```

Run:

```bash
python main.py
```
