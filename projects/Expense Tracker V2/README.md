# 💰 Phase 02 Project: Expense Tracker V2

A command line Expense Tracker that allows user to record and maintain their expenses. 

This project is an upgrade from my previous Expense Tracker. I've built this project after learning the basics of OOP. 

This project is part of my **Road to Programmer** Journey.

---

## ✨ Features

- Add Expenses
- View Expenses
    - View Expenses by Category
    - View Total Expense
- Edit Expenses
- Delete Expenses

---

## 🧠 Concepts Practiced

- OOP
- JSON
- File Handling
- Exception Handling
