# 😂 Random Joke Generator

A simple command-line Python application that fetches a random joke from an online API using the `requests` library.

This project was built while learning Python Virtual Environments (`venv`), package management with `pip`, HTTP requests, and working with JSON data.

## Features

- Fetches a random joke from an online API
- Displays the joke setup and punchline
- Uses the `requests` package
- Demonstrates API communication in Python

## Technologies Used

- Python 3
- requests
- Virtual Environment (venv)

## Installation

1. Clone the repository.
2. Navigate to the project folder.
3. Create a virtual environment:

```bash
python -m venv venv
```

4. Activate it.

Windows:

```bash
venv\Scripts\activate
```

5. Install the required packages:

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
