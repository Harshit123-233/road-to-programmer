# 💰 Expense Tracker

A command-line Expense Tracker built in Python to help users record, organize, and analyze their daily expenses.

This project was created while learning Python fundamentals and focuses on practicing dictionaries, functions, loops, and problem-solving through a practical application.

---

## ✨ Features

- Add new expenses
- View all recorded expenses
- Search expenses by:
  - Date
  - Category
- Display total expenditure
- View spending grouped by category
- Delete:
  - Entire expense history
  - Expenses from a specific date
  - Individual expense entries
