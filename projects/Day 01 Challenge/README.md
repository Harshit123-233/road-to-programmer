# 🌱 Day 01 Challenge

My very first Python challenge.

This simple program prompts the user to enter their name, age, and favorite programming language, then displays the information using formatted strings.

Although the project is small, it marks the beginning of my journey toward learning Python and becoming a software developer.

---

## ✨ Features

- Accepts user input
- Stores data in variables
- Uses formatted string literals (f-strings)
- Displays personalized output

---

## 🧠 Concepts Practiced

- Variables
- User Input
- Data Types
- Type Conversion
- f-Strings
- `print()` Function
