# 🎓 Student Record Manager

A command-line Student Record Manager built in Python to maintain student records, manage marks, and generate simple academic summaries.

This project was created while learning Python fundamentals and focuses on practicing functions, lists, dictionaries, sorting, lambda expressions, and modular problem-solving.

---

## ✨ Features

- Add new students
- View all students
- Search students by name
- Update student marks
- Delete student records
- Display the class topper
- Generate a summary report including:
  - Average marks
  - Highest marks
  - Lowest marks
  - Students ranked by marks
